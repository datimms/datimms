Usage Instructions

In the box, enter the Jira ID of the feature branch (eg IN30-1234).
Click the button or press enter key.
The OU link will open in a new tab and the URL will be saved under history in descending order of last access.
By default, 5 items will be stored (cannot currently be changed in this demo version).
If you run the page from C: drive, the page will save max 5 items you type in until you refresh. Therefore it's recommend you run this from a local web server.
If you run the page from a web server, the page will save max 5 items and will remember between sessions.
If you click a link in history, it will move to the top of the list.
The Clear List button will empty the history.

NB You may need to adjust your browser privacy settings if you have particularly aggressive settings. Please enable javascript and web storage.
